Om oss

Vi lager nettbaserte læringsverktøy og spill skreddersydd for norsk skole og folkehøyskole.

Vi kombinerer pedagogisk kunnskap med moderne webutvikling for å skape engasjerende og praktiske løsninger som faktisk fungerer i klasserommet.

Målet vårt er å gjøre læring mer motiverende – både gjennom gamification og smarte administrative verktøy som EduChoice.

Study Quest

Gamifisert læring • v2.0

Interaktivt quiz-spill med 5 fag (Matte, Norsk, Engelsk, Samfunnsfag, Naturfag). Inneholder streak-system, XP, lokale highscores og forklaringer etter hvert svar. Utviklet etter norsk læreplan.

HTML + Tailwind Vanilla JS localStorage
Prøv demo GitHub

EduChoice

Praktisk verktøy • Folkehøyskole

Smart allokeringsverktøy for folkehøyskoler. Lar administratorer definere valgfag, dager og elevpreferanser, og kjører 100 simuleringer for å finne den mest rettferdige fordelingen. Utviklet for reell bruk i norsk folkehøyskole.

HTML + Tailwind Vanilla JS Simuleringsalgoritme
Prøv demo GitHub

Hva vi kan hjelpe dere med

Vi lager skreddersydde nettbaserte læringsverktøy og administrative systemer for skoler og folkehøyskoler.

Gamifiserte læringsverktøy

Interaktive quiz, spill og læringssystemer med streak, XP og belønninger – tilpasset norsk læreplan.

Administrative verktøy

Systemer for timeplanlegging, valgfagfordeling, elevadministrasjon og rapportering (som EduChoice).

Rask prototyping

Vi kan raskt bygge fungerende prototyper og demoer som dere kan teste med elever og lærere.
